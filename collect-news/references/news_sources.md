# AI新闻源映射到四层框架

本文档将各类AI新闻源映射到四层框架，包含中文和英文新闻源。

## 新闻源分级体系

- **Tier 1**: 主要新闻源，每日必查
- **Tier 2**: 公司官方博客，每周必查
- **Tier 3**: 研究源，按需查询
- **Tier 4**: 专业细分源，特定主题查询

## 新闻源使用策略

针对每一层选择3-5个源：

**E层（变革层）**:
1. 机器之心（中文）
2. 量子位（中文）
3. MIT Tech Review（英文）
4. OpenAI Blog（英文）
5. Anthropic News（英文）

**S层（创业层）**:
1. 36氪（中文）
2. 投资界（中文）
3. TechCrunch AI（英文）
4. VentureBeat AI（英文）

**U层（使用层）**:
1. 少数派（中文）
2. InfoQ AI（中文）

**M层（奇迹层）**:
1. MIT Tech Review（英文）

### 深度模式
- 增加 Tier 3 研究源（E层）
- 增加 Tier 4 专业源（所有层）
- 执行更多搜索查询

### 快速模式
- 每层仅选择2个最权威的源
- 减少搜索查询数量

## 新闻源优先级

### 权威性排序
1. **官方博客** (OpenAI, Google, Anthropic等)
2. **主流科技媒体** (MIT Tech Review, TechCrunch)
3. **专业AI媒体** (机器之心, 量子位)
4. **财经媒体** (36氪, 投资界)
5. **社区和论坛** (Hacker News, Reddit)

### 时效性排序
1. **实时新闻** (Twitter/X, 新闻网站)
2. **每日更新** (科技媒体)
3. **每周更新** (公司博客)
4. **研究论文** (arXiv, 会议)

## 中英文内容处理

### 中文源优先
对于中国用户，优先使用中文新闻源：
- 机器之心、量子位（技术）
- 36氪、投资界（商业）
- 少数派、InfoQ（应用）

### 英文源翻译
英文内容需要：
1. 标题翻译为中文
2. 摘要翻译为中文
3. 关键点翻译为中文
4. 保留原文链接

### 混合策略
- 重大国际新闻：英文源 + 中文翻译
- 中国本土新闻：中文源优先
- 技术细节：保留英文术语
