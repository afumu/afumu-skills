# AI四层框架详解

基于《重新理解人工智能》文档的四层AI框架体系。

## 框架概述

我们在聊AI到底在聊什么？总结一下，其实一共就四层：

1. **M. 奇迹层 (Miracle Layer)**: 关于AI长期的未来思考，甚至科学幻想
2. **U. 使用层 (Usage Layer)**: 利用AI技术和工具，来降本增效
3. **S. 创业层 (Startup Layer)**: 使用AI核心技术创业，创造新的智能服务
4. **E. 变革层 (Evolution Layer)**: 底层的技术研究和支撑基础的升级、变革

## E. 变革层 (Evolution Layer)

### 定义
底层的技术研究和支撑基础的升级、变革。

### 关键特征
- 技术突破和创新
- 模型架构改进
- 开源项目和工具
- 学术研究论文
- 基础设施升级

### 典型内容
- 新模型发布（GPT-5, Claude 4, Gemini 2.0等）
- 新架构提出（Transformer变体、新的注意力机制）
- 开源项目（LLaMA, Mistral, DeepSeek等）
- 研究突破（多模态理解、推理能力提升）
- 技术论文（arXiv, NeurIPS, ICML等）

### 关键词列表
**英文关键词：**
- model release, new model, model announcement
- research paper, research breakthrough, scientific paper
- open source, open-source model, open weights
- architecture, transformer, attention mechanism
- technical advancement, breakthrough, innovation
- training method, fine-tuning, RLHF
- benchmark, evaluation, performance
- infrastructure, compute, GPU, TPU

**中文关键词：**
- 模型发布、新模型、模型更新
- 研究论文、技术突破、科研成果
- 开源、开源模型、开放权重
- 架构、技术架构、模型架构

- 训练方法、微调、强化学习
- 基准测试、性能评估
- 基础设施、算力、GPU

### 示例新闻
1. "OpenAI发布GPT-5，推理能力提升10倍"

## 分类决策树

当遇到一篇新闻时，按以下顺序判断：

```
1. 是否包含AGI/ASI/伦理/政策关键词？
   → YES: 奇迹层 (M)
   → NO: 继续

2. 是否关于技术突破/模型发布/研究论文？
   → YES: 变革层 (E)
   → NO: 继续

3. 是否关于融资/产品发布/公司并购？
   → YES: 创业层 (S)
   → NO: 继续

4. 是否关于工具使用/教程/案例/技巧？
   → YES: 使用层 (U)
   → NO: 默认归入最相关的层
```

## 多层次文章处理

有些文章可能跨越多个层次，例如：

**示例1**: "OpenAI发布GPT-5，估值突破2000亿美元"
- **主要层次**: E（变革层）- 因为核心是技术发布
- **次要层次**: S（创业层）- 涉及公司估值

**示例2**: "某AI安全创业公司获得1亿美元融资"
- **主要层次**: S（创业层）- 核心是融资
- **次要层次**: M（奇迹层）- 涉及AI安全

**处理原则**:
1. 确定一个主要层次（primary_layer）
2. 可选标注次要层次（secondary_layers）
3. 文章只在主要层次中出现，避免重复

## 相关性评分指南

### 0.9-1.0 (极高相关性)
- 重大技术突破
- 大额融资（>1亿美元）
- 主流公司重大产品发布
- 重要政策法规出台

### 0.7-0.89 (高相关性)
- 显著技术进展
- 中等融资（1000万-1亿美元）
- 知名公司产品更新
- 重要行业报告

### 0.5-0.69 (中等相关性)
- 常规技术更新
- 小额融资（<1000万美元）
- 一般产品发布
- 行业动态

### <0.5 (低相关性，过滤)
- 边缘相关内容
- 过时新闻
- 重复内容
- 营销软文

## 重要性评估指南

### High (高重要性)
- 行业标志性事件
- 改变游戏规则的技术
- 超大额融资（>5亿美元）
- 重大政策变化

### Medium (中等重要性)
- 显著但非革命性的进展
- 常规融资和产品发布
- 行业趋势报告

### Low (低重要性)
- 小规模更新
- 边缘新闻
- 次要公司动态
